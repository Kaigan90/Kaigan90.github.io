## Project description
This project is a multi-tool companion application for Fallout 76, designed to improve how players interact with the game’s underlying systems.

At its core, the application addresses a common problem players are often required to make decisions without full visibility into character builds, world information, and time based activities. This project solves that by centralizing and exposing those systems in a single, structured interface.

The application consists of three primary components:

A build simulator that allows users to allocate SPECIAL stats, equip perk cards, assign legendary perks, and apply mutations all while viewing the full combined effects in one place.

A cartography index that enables users to create and manage custom map markers with categorized notes, reducing reliance on memory or external lookups.

A daily activity tracker that consolidates multiple in-game timers and overlays activity locations, allowing for more efficient planning and execution.

Together, these tools transform fragmented and partially hidden game systems into something visible, organized, and actionable.

The result is a system that reduces friction, improves decision-making, and enhances overall player experience through better access to information.
